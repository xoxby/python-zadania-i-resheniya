## Как работать

1. Открой файл модуля в редакторе (VS Code, PyCharm, или просто IDLE)
2. Читай задание (оно написано комментарием)
3. Пиши свой код ПОД заданием (где написано `# Твой код:`)
4. Запускай: `python модуль_01_переменные.py`
5. Если ошибка — внимательно прочти текст ошибки
6. Если не понимаешь — скинь мне код, разберём вместе!
