# Домашние задания Python (курс)

Структура как в репозитории-примере: папка **`DZNN`**, внутри **`main.py`** и условия в **`.pdf`** (где выдавались).

ДЗ №1 (окружение) и №2 (Git) кода не содержат — папок под них здесь нет.

## Запуск

```bash
cd domashnie_zadaniya/DZ10
python main.py
```

или из корня репозитория:

```bash
python domashnie_zadaniya/DZ10/main.py
```
